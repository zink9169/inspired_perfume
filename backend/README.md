# Perfume Store Backend API

A complete Node.js/Express backend for a perfume e-commerce store with admin panel.

## Features

- **Product Management**: View perfumes with 10ml/35ml size options
- **Order System**: Customers can order perfumes with personal details
- **Admin Panel**: Manage orders, products, and view statistics
- **Authentication**: JWT-based auth for admin access
- **Neon PostgreSQL**: Cloud database setup
